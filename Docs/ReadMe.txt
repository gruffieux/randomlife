****************************************************** Random Life ********************************************************

Random Life est un jeu de rôle solo en 2D. Il fonctionne sur Windows et nécessite DirectX.


Actions principales (Lancez GameConfig.exe pour configurer les touches)
_______________________________________________________________________


- Attaquer 			-> Cibler l'adversaire en cliquant dessus -> Presser la touche au clavier (Enter)
- Lancer un sort		-> Sélectionner un sort depuis la liste des sorts -> Mémoriser le sort -> Cibler l'adversaire (pour les sorts destructeurs) -> Presser la touche (Pavé numérique 3)
- Dormir			-> Presser la touche (Pavé numérique 1)
- Prendre un objet au sol	-> Se positionner sur l'objet -> Presser la touche au clavier (Enter) -> Cliquer le bouton 'Prendre'
- Passer le tour		-> Presser la touche au clavier (Barre espace)


Actions sur un personnage
_________________________

- Espionner	-> Cibler -> Cliquer le bouton 'Espionner' (visible uniquement sur les persos ennemis)
- Voler		-> Cibler -> Espionner -> Sélectionner un objet de l'inventaire -> Cliquer le bouton 'Voler' (ennemis uniquement)
- Acheter	-> Cibler un Marchand -> Cliquer le bouton 'Commercer' -> Sélectionner un objet de l'inventaire du marchand -> Cliquer le bouton 'Acheter'
- Vendre	-> Cibler un Marchand -> Cliqer le bouton 'Commercer' -> Sélectionner un objet de votre inventaire -> Cliquer le bouton 'Vendre'


QUelques touches pratiques (non configurable)
_____________________________________________

Configuration					-> F3
Mode fenêtre / Plein écran			-> F4
Mettre en pause ou revenir au menu principal	-> Esc
Chargement rapide				-> F6
Sauvegarde rapide				-> On peut pas, désolé... mais le jeu est sauvé à chaque fois que vous augmentez vos compétences ou finissez une mission.


Spécialités des armes
_____________________

Lorsque un coup critique est porté à l'adversaire, un bonus est déclanché. Ce bonus varie en fonction du type d'arme:

- Main nue et gourdin	-> L'adversaire est sonné et passe un tour
- Epée			-> Les dégats infligés sont doublés
- Hache			-> Le coup traverse les armures, les dégats infligés ne sont donc pas réduits


Conseil pour débuter
____________________

Regardez chaque fenêtre depuis la barre de raccourcis en bas à droite.
Etudiez bien la fenêtre 'Personnage'. Si vous passez la souris sur les compétences, une bulle montre les caractéristiques associées. Ce sont ces caractéristiques que vous pourrez monter lorsque vous passez à un niveau supérieur.
Les caractéristiques secondaires sont calculées automatiquement. Passez dessus avec la souris pour voir la formule.
La caractéristique 'Vitesse de combat' est indispensable pour avoir plus d'actions par tour lors d'un combat. Regardez bien sa formule et montez les compétences nécessaires pour avoir au moin 2 actions par tour.


Voilà quelques bases, cela devrait suffire pour bien commencer... A vous de découvrir le reste!
